# CASE 001 — AI Hallucination

## Object

~~~text
x = answer generated by an AI system
~~~

## Framework

~~~text
F = language-model probability framework
~~~

## Use

~~~text
U = user treats the answer as factual, legal, medical, technical, or historical ground
~~~

## Local Closure

~~~text
LocalClosureF(x) = the answer is fluent, coherent, and presented as sufficient
~~~

This can be locally valid inside the framework of linguistic continuation.

## Valid Scope

~~~text
ValidScope(F,x) = linguistic plausibility, pattern completion, internal coherence
~~~

## Scope Violation

~~~text
ActualUseU(F,x) exceeds ValidScope(F,x)
~~~

The answer is used as externally verified fact.

## Critical Excluded Dependence

~~~text
d = source verification, date, jurisdiction, measurement, evidence, domain constraint
~~~

~~~text
CriticalDepΩ(d,x,U) ∧ ExcludedF(d)
~~~

The framework does not itself guarantee the required external dependency.

## Diagnosis

~~~text
ErrΩ(x,F,U) = yes
~~~

The error is not that the answer exists.

The error is that a locally closed linguistic object is used beyond the field in which its closure is valid.

## Correction

~~~text
Declare scope.
Preserve uncertainty.
Require source-grounded verification when the use depends on external facts.
~~~
