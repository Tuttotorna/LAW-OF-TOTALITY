# Templates

This directory contains reusable audit templates for applying the Law of Totality.

The purpose of a template is not to explain the whole theory again.

The purpose is to make the framework usable on real objects:

- AI answers;
- theories;
- models;
- protocols;
- business decisions;
- engineering plans;
- medical or legal recommendations;
- complex projects.

The main template is:

- [`STRUCTURAL_ERROR_AUDIT_TEMPLATE.md`](STRUCTURAL_ERROR_AUDIT_TEMPLATE.md)

A valid audit should be able to reach one of three verdicts:

1. structural error;
2. no structural error;
3. insufficient evidence.

If an audit cannot produce negative results, it is too broad.

If it cannot state what would defeat the diagnosis, it is too weak.
